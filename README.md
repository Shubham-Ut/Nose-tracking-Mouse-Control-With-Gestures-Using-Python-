# Nose-tracking-Mouse-Control-With-Gestures-Using-Python-
Description:- This project is an AI virtual mouse system that lets you control your computer using head movement, eye blinks, and mouth movement through a webcam. It uses MediaPipe Face Landmarker to track facial points, OpenCV to capture video, and PyAutoGUI to control the mouse.

Move head → Move cursor

Short blink → Left click

Long blink → Right click

Open mouth → Drag and drop

It is useful for **hands-free computer control, accessibility, and computer vision projects
